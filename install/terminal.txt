# import terminal profile

- open terminal
- Preferences
- Tab: Profiles
  - bottom left settings: import..
  - choose: `~/dotfiles/zsh/jellybeans.terminal`
- Tab: General
  - *On startup, open: New window with profile:* **jellybeans**
  - *New windows open with: same Profile / same Working Directory*
